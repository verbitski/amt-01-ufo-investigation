# DEC-### — Decision title

- **Status:** proposed / accepted / rejected as primary / deprioritized / parked / paused / superseded
- **Recorded date:** YYYY-MM-DD
- **Scope:** case / analysis / hypothesis / model / evidence / research / governance
- **Supersedes:** none or DEC-###
- **Superseded by:** none or DEC-###
- **Affected canonical documents:**

## Context

Describe the problem, correction, conflict, or scope change that required a decision.

## Inputs and evidence considered

List the observations, estimates, derivations, external sources, or prior decisions that materially affected the choice.

## Alternatives considered

1. Alternative A
2. Alternative B
3. Alternative C

## Decision

State the chosen outcome unambiguously.

## Rationale

Explain why this option best fits the retained evidence, project rules, and current objectives.

## Consequences and limitations

Document what changes, what remains unresolved, and any new risks or dependencies.

## Review trigger

State the evidence or condition that would require this decision to be revisited.
