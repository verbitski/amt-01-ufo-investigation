# Documentation guide

AMT-01 is a witness-led investigation of an unidentified experience reported near the Antalya coast. Here, **UFO** means only that the witness has not identified what was seen. It is not a claim about origin, operators, or extraordinary technology.

## Evidence boundary

The available case material consists of:

- one retrospective first-person witness record;
- a later underside reconstruction and later digital scene reconstructions;
- dated corrections, estimates, and an uncertainty register;
- analysis and project-developed hypotheses derived from that record.

The project has no known event photograph or video, independent witness account, sensor confirmation, recovered exact site, or instrumented measurement of size, height, distance, speed, sound, or environmental effects.

The documentation can preserve and test the account. It cannot establish what caused the experience from the available evidence alone.
