# Epistemic Framework

## 1. Why this file exists

AMT-01 begins with a personally salient retrospective memory and includes highly speculative physical interpretations. The project will fail if those layers are merged. This framework preserves the distinction.

## 2. Evidence classes

### Direct observation (`OBS`)

A feature the witness remembers perceiving directly, such as shape or the fact that no obvious sound or downwash was noticed.

### Estimate (`EST`)

A scale inferred visually: altitude, length, distance, duration, or speed.

### Derived result (`DER`)

A mathematical consequence of stated estimates and formulas.

### Inference (`INF`)

An interpretation such as “the object behaved as though it had substantial inertia.”

### Hypothesis (`HYP`)

A proposed mechanism such as distributed lift, vacuum exclusion, or a spectrally selective sensor window.

### External report (`EXT`)

A claim found in a public record. Its presence in a government file establishes that the report exists, not that the reported interpretation is correct.

### Decision (`DEC`)

A project-scope or model-management choice.

## 3. Negative observations

Statements such as “no sound” and “no downwash” are retained as observation facts in the case baseline, with this technical interpretation:

> No such signature was detected by the witness under the stated conditions.

They do not establish an infinitely precise zero.

## 4. Confidence and uncertainty

### Higher-confidence memory record

Repeatedly stated, visually salient, and central to the witness's recollection. This describes memory stability or salience, not the probability that the external event occurred exactly as recalled:

- triangular shape;
- sharp nose;
- flat underside;
- silence;
- straight, level motion;
- very low altitude;
- very dark surface;
- regions remembered as completely black.

### Medium-confidence estimate

- 6–8 m length;
- 2.5–3 m width;
- 4–5 m altitude;
- 40–50 m path in 12–15 seconds;
- 15–20 m lateral distance;
- roughly 5–7 completely black regions.

### Explicitly uncertain

- exact thickness;
- exact upper shape;
- exact count, position, and size of black regions;
- exact day, month, hotel, and location coordinates;
- precise object mass;
- whether black regions were openings, windows, or surface materials.

## 5. Model discipline

A model may be compatible with the observation without being supported by it.

Example:

```text
Observation: no downwash was noticed.
Inference: conventional rotor support is disfavored.
Hypothesis: weight was supported by a field-like distributed interface.
```

Only the first line is an observation. The second is an engineering inference. The third remains speculative.

## 6. Conservation-law rule

The project does not use phrases such as “momentum disappears” or “physics is violated” as explanations.

Any viable model must specify where momentum, energy, pressure stress, and entropy go, even if the mechanism is unknown.

## 7. External-source rule

Priority order:

1. original sensor data or original government record;
2. original PDF/image/audio/video;
3. official summary or metadata;
4. user-created Markdown transcription;
5. secondary journalism;
6. social-media retelling.

AI-assisted OCR is a search layer, not the final authority for exact wording.

## 8. Model update rule

A model status changes only when one of the following occurs:

- a new AMT-01 fact is added or corrected;
- a calculation invalidates a required mechanism;
- a public case provides a strongly constrained physical match;
- a model produces a distinctive test and fails it;
- a simpler model explains the same signature set.
