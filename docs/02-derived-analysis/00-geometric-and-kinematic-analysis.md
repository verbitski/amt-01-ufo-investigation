# Geometric and Kinematic Analysis

## 1. Input ranges

```text
Length L = 6–8 m
Rear width W = 2.5–3 m
Later estimated path d = 40–50 m
Duration t = 12–15 s
Altitude h ≈ 4–5 m
Lateral observer distance R ≈ 15–20 m
```

## 2. Speed


Average speed:

\[v = \frac{d}{t}\]

\[v \approx \frac{40-50}{12-15} = 2.7-4.2\;\text{m/s}\]

\[v \approx 9.6-15\;\text{km/h}\]

This is scenario mathematics, not a measurement of speed. Path length and duration are correlated retrospective estimates. If the current estimates are approximately correct, they describe motion near running speed and create useful comparison questions about how a form of the remembered scale might remain aloft.

## 3. Planform area

Approximating the underside as a triangle:

\[S = \frac{1}{2}LW\]

\[S \approx 7.5-12\;\text{m}^2\]

\[S_{nominal} \approx 9.6\;\text{m}^2\]

## 4. Geometric center

For a uniform triangular plate, the centroid lies approximately two-thirds of the distance from the nose to the rear edge:

\[x_c=\frac{2}{3}L\]

For a nominal 7 m length:

\[x_c\approx4.7\;\text{m from the nose}\]

This is relevant only to a conditional model of a roughly uniform triangular body. A uniformly distributed support system would place its resultant near this point, but no such system or mass distribution has been established. The black-region reconstruction is too approximate to justify a detailed correlation with the centroid.

## 5. Approximate angular scale

Under the current 15–20 m lateral-distance estimate, a 6–8 m length would subtend an angular extent of roughly 17–30°. A small-angle approximation is inappropriate; use:

\[\theta=2\arctan\left(\frac{L}{2R}\right)\]

For example, 7 m at 17.5 m gives an angular extent of approximately 22.6°. If the size and range estimates are jointly correct, the form would have occupied a substantial part of the witness's view. This calculation does not independently validate either estimate.

## 6. Error sensitivity

The speed estimate depends linearly on path length and inversely on duration. The area estimate depends linearly on both length and width. Size, range, altitude, and path length were reconstructed from the same memory and should not be treated as independent measurements. Mass-dependent calculations remain scenario analyses because mass is unknown.
