# Public Source Corpus

## 1. Official PURSUE source

In a project snapshot checked on 2026-07-19, the U.S. Department of War PURSUE site listed four release tranches, with Release 04 dated July 10, 2026, and displayed 334 files. This is a dated snapshot of a changing website, not a canonical corpus count. A future dataset release must preserve a manifest of exactly which records were screened.

Primary source:

- https://www.war.gov/UFO/

Important interpretation:

> “Unresolved” means the releasing authority has not made a definitive determination from available data. It does not mean extraterrestrial, anomalous physics, or even necessarily a physical craft.

## 2. UFO-USA Markdown corpus

Repository:

- https://github.com/DenisSergeevitch/UFO-USA

Provisional discovery role:

- AI-assisted Markdown transcription can support search and triage;
- mutable `main`-branch pages are not stable evidence locators;
- exact wording, numbers, classification, and context must be checked against the original government PDF, image, audio, or video;
- corpus-size claims require a pinned commit and frozen manifest before publication as dataset facts.

## 3. Source hierarchy for this project

1. Original PURSUE PDF/image/video/audio.
2. Original government metadata.
3. UFO-USA Markdown transcription for search and triage.
4. Secondary reporting only for discovery.

## 4. Corpus-search strategy

Do not begin with “alien” or “extraterrestrial,” because those terms retrieve cultural, policy, magazine, and contactee material mixed with operational reports.

Prioritize physical signatures:

```text
triangular / metallic / dark / smooth
no visible propulsion
no sound / no trail / no exhaust
hover / sudden acceleration / direction change
low altitude / close range
water / ocean / transmedium
trained observer / sensor / coordinates / speed / altitude
```

## 5. Corpus limitation

A government archive can contain:

- operational reports;
- witness statements;
- unresolved sensor events;
- rumors;
- public correspondence;
- speculative proposals;
- newspaper clippings;
- contactee literature;
- later historical compilations.

Document custody must not be confused with validation of every enclosed claim.

## 6. Admission rule

A comparison case enters a reviewed dataset release only when it has:

1. an original-source URL or preserved rights-cleared copy;
2. an exact page, timecode, frame, or record locator;
3. a human verification state;
4. a source-quality assessment;
5. an inclusion reason and discovery query;
6. known mundane alternatives and contradictions;
7. rights and privacy fields;
8. a corpus-snapshot identifier.
