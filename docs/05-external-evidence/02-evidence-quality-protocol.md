# Evidence Quality Protocol

An evidence tier describes the available record, not witness sincerity, strangeness, truth, or probability of an extraordinary origin. A case can be interesting and still have a weak evidence package.

## 1. Evidence tiers

### Tier A — Strong physical constraint

- multiple synchronized sensors or cameras;
- triangulated distance;
- calibrated timing/FOV;
- preserved original media;
- known platform/sensor metadata;
- observable environmental interaction or expected interaction clearly absent.

### Tier B — Constrained single-source case

- one camera/sensor or trained multi-witness observation;
- distance constrained through occlusion, geometry, radar, cloud layer, or landmarks;
- original source preserved;
- plausible mundane explanations actively tested.

### Tier C — Descriptive report

- credible witness or government report;
- useful morphology/behavior;
- insufficient range/speed/sensor information.

### Tier D — Discovery lead

- social-media clip;
- one-frame streak;
- unknown metadata;
- contactee literature;
- newspaper clipping without primary testimony;
- AI transcription without original-page verification.

## 2. Comparative-case scoring

Add value for:

- trained observer;
- close range;
- multiple independent witnesses;
- sensor pairing;
- known coordinates, altitude, and speed;
- no visible propulsion;
- no trail/downwash/noise under conditions where they should be detectable;
- transmedium interaction with constrained geometry;
- original data availability.

Subtract value for:

- uncertain distance;
- single frame;
- rolling-shutter/compression risk;
- near-lens insect possibility;
- obvious wind-borne behavior;
- derivative retelling;
- contactee or promotional context;
- contradictions inside the source.

## 3. “Government file” rule

A file can document that an official agency collected or discussed a report. It does not automatically authenticate the object's interpretation.

## 4. OCR rule

Search converted Markdown first, then verify exact statements against the original PDF page before citing or calculating from them.

## 5. AMT-01 evidence status
